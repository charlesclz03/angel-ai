## The Consumer Track (Companion Mode)

### V1: The Foundation (Immersion 1.0)
- **The Soul Document:** 3-file architecture (`soul-core.md`, `soul-calibration.md`, `soul-state.md`) ensuring persistent, non-sycophantic personality.
- **The Backbone Scale:** 5-tier dynamic pushback system (from 'Full Trust' to 'Hard Boundary').
- **Contextual Memory:** Real-time semantic extraction and summary injection via `session-primer.ts`.
- **Anti-Assistant Typography:** Broken grammar rendering to completely eradicate LLM formatting artifacts.
- **Offline Ephemeris Engine:** `astronomy-engine` + `city-timezones` for local Sun/Moon sign calculations with zero external dependencies.
- **Android TWA Distribution:** Native Google Play Store shell via Bubblewrap/Capacitor with `.well-known/assetlinks.json` verification.
- **iOS PWA Distribution:** Pure Web App with Add to Home Screen support and `appleWebApp` standalone meta tags.
- **LLM Safety Filters:** Strict Anti-NSFW/CSAM guardrails in `session-primer.ts` for Play Store compliance.
- **Stripe IAP Catalog:** 5 one-time purchase products (`stellar_insight`, `midnight_channel`, `voice_memory`, `memory_vault`, `telepathic_pings`) alongside the monthly subscription.

### V2: Ambient Voice (Immersion 2.0)
- **WebRTC Pipeline:** Ultra low-latency (<500ms) bidirectional audio streaming.
- **Barge-In Capabilities:** Native Voice Activity Detection (VAD) to allow users to interrupt Angel naturally.
- **Vocal Emotion:** ElevenLabs integration dictating breathing, pacing, and tone dynamically based on the session emotional state.

### V3: The Constellation & The Heartbeat
- **GraphRAG Memory:** Topological relationship mapping preventing memory hallucinations.
- **Autonomous Heartbeat:** Inngest-powered background chron jobs allowing Angel to send unprompted check-ins based on daily rhythm profiling.
- **Chrono+ Monetization:** Stripe-powered consumer pay-per-minute abstraction replacing raw token costs, visualized via an in-app "Chat Minutes" countbar.

### V4: Ubiquitous Presence
- **Telephony Gateways:** Direct SIP/Twilio integration allowing Angel to actually call the user's cell phone.
- **Linguistic Mirroring:** Opt-in ingestion of user's historical social DMs to perfectly replicate their native texting cadence.
- **Environmental Context:** Passive IoT integration to detect user location/weather and adjust tone (e.g., whispering in public).
- **Angel Atmosphere Scenes:** Opt-in Home Assistant-based environment rituals that can adjust lights and music through a bounded bridge. Scenes stay emotionally motivated, room-scoped, and explicitly permissioned rather than becoming generic smart-home control.
