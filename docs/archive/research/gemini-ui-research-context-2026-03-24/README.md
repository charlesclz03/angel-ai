# Gemini UI Research Context Package

This folder packages the Angel AI context files for a Gemini Advanced research pass on sci-fi-inspired companion UI direction.

## Use These Folders

- `required/`
- `strongly-recommended/`
- `extra-visuals/`

Prefer the nested copies that preserve the original relative paths, for example:

- `required/docs/README.md`
- `required/components/organisms/AngelChat.tsx`
- `strongly-recommended/app/chat/page.tsx`
- `extra-visuals/docs/design/Angel AI Design from Stich/stitch/angel_ai_landing_page/screen.png`
