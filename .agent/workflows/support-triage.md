$ARGUMENTS

## Task

Use the `support-ticket-triage` skill at `.agent/skills/support-ticket-triage/SKILL.md`.

1. summarized issue
2. likely subsystem owner
3. likely cause or hypotheses
4. reproduction and verification path
5. backlog or handoff changes if priority shifts
